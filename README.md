# Lelo's Catering

A responsive single-page website for Lelo's Catering, built with Next.js and ready to deploy on Vercel.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy to Vercel

1. Push this project to the Lelo's GitHub repository.
2. Import the repository at [vercel.com/new](https://vercel.com/new).
3. Keep the detected **Next.js** framework preset and default build settings.
4. Select **Deploy**.

No environment variables are required.
